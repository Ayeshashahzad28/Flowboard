Flowboard is a modern **Kanban-style task management system** built with **React (Vite)**.  
It simulates real-world SaaS productivity tools like **Trello / Jira** using a fully frontend architecture.

> ⚠️ **Important:** This is a demo project only. Not production-ready. No backend, no secure authentication.

---

# 🧠 Overview (What this project is)

Flowboard demonstrates how a **real SaaS product UI** can be built using only frontend technologies:

- Task lifecycle management
- Role-based access control
- Simulated collaboration system
- Time tracking engine
- Persistent local storage system

It is designed for **learning, portfolio, and interview demonstration purposes**.

---

# ✨ Features

## 📌 Task Management System
- Create / edit / delete tasks
- Drag & drop across Kanban columns
- Subtasks with completion tracking
- Priority + due date system

## 👥 Role-Based Access
- 🟣 Owner → Full control
- 🔵 Editor → Task management
- 🟢 Viewer → Read-only access

## ⏱️ Time Tracking
- Start / pause task timer
- Total time tracking per task
- Session persistence (localStorage)

## 🤖 Simulated Collaboration
- Bot users simulate activity
- Task locking system
- Real-time toast notifications
- Presence indicators

## 💾 Data Persistence
- Fully powered by browser `localStorage`
- No backend dependency
